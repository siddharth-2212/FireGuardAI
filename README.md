##  Overview

FireGuardAI is designed to simulate and monitor fire risk across multiple zones using sensor telemetry data such as temperature, smoke, and CO levels.

The system processes data, predicts fire risk using ML logic, and displays actionable insights through a modern dashboard interface.

---

##  Features

### Real-Time Dashboard
- Active sensor monitoring
- System health overview
- ML accuracy tracking

### Smart Alerts System
- Critical alert detection
- Acknowledge alerts functionality
- Severity-based classification

###  Sensor Network Visualization
- Live telemetry display
- Risk level indicators (Low, Medium, High)
- Zone-based monitoring

### ML Detection Engine
- Predicts fire probability
- Confidence scoring
- Risk classification

### Activity Feed
- Tracks recent system events
- Real-time updates

### Architecture
- Designed for IoT integration
- Scalable backend
- API-driven system
